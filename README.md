# Python Mini Projects 🐍

Bu depo (repository), Python ile geliştirilmiş ufak çaplı, pratik ve eğlenceli projeleri barındırmaktadır. Öğrenme, yeni kütüphaneler deneme ve çeşitli problemleri çözme amacıyla bu reypositoriye **sık sık yeni ufak projeler eklenecektir.**

*(This repository contains small, practical, and fun projects developed with Python. **New mini-projects will be added frequently** to explore new concepts and libraries.)*

## 📂 Mevcut Projeler / Current Projects

Şu an depoda bulunan projeler:

- **[ASCII_Converter](./ASCII_Converter)**: Görüntüleri (veya metinleri) ASCII sanatına dönüştüren araç.
- **[earthquake_data](./earthquake_data)**: Deprem verilerini internet üzerinden çeken, işleyen ve gösteren uygulama.
- **[md_reader](./md_reader)**: Markdown (`.md`) dosyalarını okumak veya Terminal'de göstermek için temel bir araç.
- **[wordle](./wordle)**: Popüler kelime tahmin oyunu Wordle'ın Python ile geliştirilmiş klonu.

*Her projenin kendi klasöründe daha detaylı açıklama veya kullanım talimatları bulunabilir.*

## 🚀 Kurulum / Installation

Bu depodaki projelerin ortak bağımlılıkları ana dizindeki `requirements.txt` dosyasında bulunabilir. Kütüphaneleri kurmak için aşağıdaki komutu kullanabilirsiniz:

```bash
pip install -r requirements.txt
```

*(Bazı projelerin kendine özel gereksinimleri olabilir, ilgili klasörleri kontrol ediniz.)*
